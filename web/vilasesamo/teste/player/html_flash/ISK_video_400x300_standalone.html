Input params (XML)

Input params (FlashVars)

  • configUrl: path to XML config
  • skinUrl: path to SWF for button skins
  • jsShareMethod: name of Javascript method called when the "send to friend" button is clicked
  • jsStatsMethod: target method to call to feed usage stats to 3rd party web stats (Google).
  • autoStart: video auto starts on launch
